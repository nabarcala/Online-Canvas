# Online-Canvas

A painting canvas accessible from the web. Draw anything you want!

[Check out the live site!](https://www.natsketch.com/)

### Features:
* Change brush color
* Change brush size
* Draw shapes:
    * Lines
    * Squares
    * Circles
* Eraser tool
* Clear the canvas
* Paint Bucket Tool
* Save the image
* Upload an image to canvas
* Undo and Redo buttons
