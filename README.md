# Online-Canvas

A canvas accessible from the web. Draw anything you want!

[Check out the live site!](https://online-canvas.netlify.app/)

### Features:
* Change brush color
* Change brush size
* Draw shapes:
    * Lines
* Eraser tool
* Clear the canvas
* Paint Bucket Tool
