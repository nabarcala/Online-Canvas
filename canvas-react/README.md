# Online-Canvas

A painting canvas accessible from the web. Draw anything you want!

[Check out the live site!](https://online-canvas.netlify.app/)

### Features:
* Change brush color
* Change brush size
* Draw shapes:
    * Lines
    * Squares
    * Circles
* Eraser tool
* Clear the canvas
* Paint Bucket Tool
* Save the image

### Upcoming Features:
* Upload image to canvas
* Undo and redo buttons
* Crop canvas image 
* Layers
